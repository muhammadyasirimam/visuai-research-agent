# 🔬 VisuAI Research Agent

**Transform Research Data Into Stunning Visual Intelligence**

Welcome to the VisuAI Research Agent documentation. This tool helps researchers visualize any kind of data into meaningful graphical representations according to their needs and requirements.

## What is VisuAI?

VisuAI is an open-source AI agent framework that automatically transforms raw research datasets into publication-ready, interactive visualizations. It understands your research domain, suggests optimal chart types, explains insights with XAI, and exports to multiple formats.

## Key Features

- 🤖 **AI Agent Orchestration** — Multi-agent system for end-to-end visualization
- 📊 **50+ Chart Types** — From basic histograms to specialized domain charts
- 🧠 **5 Research Domains** — ML, Healthcare, Finance, Social Science, Engineering
- 🔍 **XAI Integration** — SHAP, LIME, Permutation Importance
- 📄 **7 Export Formats** — HTML, PDF, LaTeX, Jupyter, PPTX, Markdown, PNG
- 💬 **Natural Language** — Chat with your data
- 🌐 **Web Interface** — Beautiful Streamlit app
