🚀 Quick Start

Get up and running in 3 lines of code

# 1. Install VisuAI
pip install visuai-research-agent

# 2. Import and initialize
from visuai import ResearchAgent

agent = ResearchAgent(domain="machine_learning")

# 3. Load data and generate visualizations
agent.load_data("your_data.csv")
result = agent.run(query="Show me feature correlations and distributions")

# 4. View and save results
result.show()
result.save("./output")
